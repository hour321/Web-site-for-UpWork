## ğŸ”§ Customization

### Colors
Edit the gradient colors in `tailwind.config.js`:

```javascript
theme: {
  extend: {
    colors: {
      // Add your custom colors here
    }
  }
}
```

### Animations
Modify animations in the component files using Framer Motion:

```typescript
<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  transition={{ duration: 0.8 }}
>
  Your content
</motion.div>
```
